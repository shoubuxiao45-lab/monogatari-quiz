# 物语系列知识水平检定 (Monogatari Quiz)

这是一个专门为《物语系列》粉丝制作的交互式答题网站。

### 🔗 快速进入
**[点击这里开始测试](https://shoubuxiao45-lab.github.io/monogatari-quiz/)**

### 🌟 项目特色
- **SHAFT 风格视觉**：还原动画经典的“黑齣/赤齣”转场。
- **硬核题库**：包含从《化物语》到《续·终物语》共 30 道深度题目。
- **调查复盘**：答题结束后可查看详细的对错分析，纠正那些“你不知道的事”。

### 🛠️ 技术栈
- HTML5 / CSS3 (Flexbox & Grid)
- Vanilla JavaScript (Web Audio API)
